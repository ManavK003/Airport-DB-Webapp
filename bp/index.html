Airport Database Management System

Welcome to my database project for managing airport data, landings, flights, airlines, and pilots.

Project Problem

This project solves the problem of efficiently storing, managing, and analyzing airport operations using a PostgreSQL database. It provides structured data storage, easy querying, and operational insights for better decision-making.

SQL Queries Used

  • Total Landed Weight by Airline:
    SELECT a.name AS airline_name, SUM(l.total_landed_weight) AS total_landed_weight
    FROM landings l
    JOIN airlines a ON l.airline_id = a.airline_id
    GROUP BY a.name
    ORDER BY total_landed_weight DESC;

  • Total Landed Weight by Airline and Flight ID:
    SELECT a.name AS airline_name, f.flight_id, SUM(l.total_landed_weight) AS total_landed_weight
    FROM landings l
    JOIN flights f ON l.flight_id = f.flight_id
    JOIN airlines a ON f.airline_id = a.airline_id
    GROUP BY a.name, f.flight_id
    ORDER BY total_landed_weight DESC;

  • Total Number of Delays by Flight and Airline:
    SELECT d.flight_id, a.name AS airline_name, COUNT(d.delay_id) AS total_delays
    FROM delays d
    JOIN flights f ON d.flight_id = f.flight_id
    JOIN airlines a ON f.airline_id = a.airline_id
    GROUP BY d.flight_id, a.name
    ORDER BY total_delays DESC;
